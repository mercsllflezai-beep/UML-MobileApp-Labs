## 🎯 Описание проекта

**HelpHub** — мобильное приложение для:
- 🆘 Экстренной психологической помощи
- 👥 Организации сообществ по интересам
- 📅 Планирования встреч и мероприятий
- 💬 Анонимного чата с модерацией

**Цель проектирования:** разработать полную UML-модель системы для последующей реализации на Python/Django + мобильный клиент.
